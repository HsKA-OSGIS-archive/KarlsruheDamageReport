User Manual:


Please find herein the user Manual of this website.


The website first opens to its home page, where the incidents report map is displayed. This map shows the locatons of the incidents that have been reported by users of the website. It is not necessary to register on the website to view these incident notifications, but should you wish to add incidents you will need to register as a user.


The home page of the website shows a map and a legend (which explains the data shown on the map). It also allows you to search the notifications in one of three ways.


1. Search by Geometry: This allows you to search by the type of geometry of the notification you are looking for. For example, if you are searching for a broken traffic light notificaton, the geometry type would be point, whereas an incident such as a road block would be a line. The geometry types are displayed in the legend.


2. Search by Type of Incident: This allows you to select the incident type of the event you are looking for.


3. Search by ID: This allows you to search via the ID of the event as it stored in the database.


The website allows for three types of user privileges, which are listed below:


1. Unregistered Users: Can view listed incident notifications.


2. Editors: May create new notifications using a form.


3. Administrators: May add, edit and delete notifications. Can add “emergency” issues.


How to register:


1. In the top right hand coner of the screen, select the 'Login' tab of the website. Then click on 'Register'.


2. A form will become visible. Fill in the form according to the given instructions. Then select 'Register'. You will then be able to log in.


3. Should the 'Register' button result in an error, adjust your details accordingly.


Now that you have registered, you can proceed to use the website further.


Note that you must be logged in to continue. To log in, select the 'Login' button in the top right corner of the web page. Fill in your details and press the log in button. Now, navigate to either the home page, the create page or the edit page by selecting the tabs in the top left of the web page. Each page corresponds to its functionality.


To create an event if you are a registered editor:


1. In the top left hand coner of the screen, select the 'Create' tab of the website. You can now select the geometry type, notification type and fill in a description of the notification.An image can be added if desired. You will find that selecting a specific event type will lead to an automatic geometry selection.


2. Press the 'Create' button at the end of the form.


3. The new event has been created and is thus visible in the table of events, as well as in the home page.


To edit an event if you are a registered administrator:


1. In the top left hand coner of the screen, select the 'Edit' tab of the website. The data can be changed in the same way as an event is created. Notifications can also be deleted or marked as solved.


2. The updated event has been created and the changes are thus visible in the table of events, as well as in the home page..
